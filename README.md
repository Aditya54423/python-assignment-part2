# Part 2: Data Structures

## Project Overview
This project implements an **Order Management System** designed to demonstrate the efficient use of Python's built-in data structures, specifically focusing on Lists, Dictionaries, and Sets.

## Features
* **Inventory Management:** Uses dictionaries to map product IDs to prices and stock levels.
* **Order Processing:** Utilizes lists to maintain the sequence of incoming orders.
* **Unique Item Tracking:** Employs sets to quickly identify unique categories or items within an order.
